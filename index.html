The Rock, Paper, Scissors Story

I am Egedegbe Oghenero, am a Software Engineer by aspiration. As a part of the journey, am not just interested in the Logic but how user interactions can interface with it; hence this project. I am interested making this game host more than user, and that is Project work in progress. Thank you, and join me as we grow!

Intro: Select Game Mode

Here you are requested to Select the desired game mode, for your gaming experience.

NB: Player versus Player mode is still work in progress.

Play: Choose your tools

This sessions allows the player to give his/her name, the number of rounds, if the game mode is Player versus AI. However, if the game mode is Player (1) versus PLayer (2) a token would be generated to host a match for 3 rounds. While Random would be with any random player. Player can then proceed to Select any option of choice.

Verdict: Match result returned

The game returns the outcome of the match, after the number of rounds have been exhausted, and ends the game play.
